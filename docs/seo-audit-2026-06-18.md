## 4. Corrections (R2 — verified against live, override specialist claims)

1. **FAQ schema EXISTS.** The technical specialist reported "/faq has zero FAQPage
   JSON-LD." Live HTML carries `"@type":"FAQPage"` + 19 `Question` entries — a
   false negative (it grepped `"@type": "FAQPage"` with a space; live HTML is
   minified, no space). Finding struck.
2. **BlogPosting schema EXISTS.** The technical specialist guessed posts "likely
   lack BlogPosting." Source confirms it is emitted (`app/blog/[slug]/page.tsx:64`).
   The real gap is the missing `image` property (H2), not the schema's absence.
3. **`coverImage` is unused.** The sitemap specialist assumed posts "use coverImage
   in frontmatter." 0/29 posts populate it — relevant to H2's fix size.
